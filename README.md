# zh-site-lab

## 项目简介

`zh-site-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 页面的集中存放与展示用途。

## 目录结构

```
├── pages/          # 存放各独立 HTML 页面的子目录
├── assets/         # 公共资源文件（CSS、JS、图片等）
├── index.html      # 仓库根目录入口页面
└── README.md       # 本文件
```

- `pages/`：每个子目录代表一个独立页面项目，包含其对应的 HTML 文件及相关资源。
- `assets/`：存放多个页面可能共用的样式、脚本或图像文件。
- `index.html`：仓库根目录的入口页面，可用于导航或展示页面列表。

## 页面归档说明

- 每个独立页面项目均以子目录形式存放在 `pages/` 下，命名遵循简短、无歧义的原则。
- 页面内容以静态 HTML 为主，可包含内联或外部的 CSS 和 JavaScript。
- 本仓库不保证所有页面均持续可用或及时更新，归档内容可能包含实验性、临时性或历史版本页面。

## 维护说明

- 本仓库由维护者不定期更新，用于记录和发布 HTML 页面存档。
- 欢迎提交 Issue 或 Pull Request 来报告问题、提出建议或贡献新页面。
- 提交新页面时，请确保：
  - 页面不与已有页面冲突或重复。
  - 页面内容不包含恶意代码、侵犯版权或违反法律法规的信息。
  - 页面资源尽量自包含，或明确引用公共 CDN 资源。
